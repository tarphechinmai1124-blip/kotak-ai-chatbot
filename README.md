# 🏦 Kotak AI Banking Assistant

An AI-powered banking chatbot built using Streamlit with a modern WhatsApp-style UI.  
It provides instant responses for common banking queries like balance check, loans, credit cards, and fraud assistance.

---

## 🚀 Features

- 💬 Chat-based interface (WhatsApp style)
- ⚡ Quick action buttons (Sidebar)
- 🧠 Smart keyword-based responses
- 🎨 Attractive modern UI with custom CSS
- 🕒 Real-time chat with timestamps
- 📱 Responsive and easy to use

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit ⚡
- HTML + CSS 🎨
